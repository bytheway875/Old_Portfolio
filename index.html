My Work

Hashtats

My final project for MakerSquare. Hashtats is a community for tattoo artists and enthusiasts to get inspiration (and find artists) for their next tattoo.

This Ruby on Rails app offers integration with the Twitter API to allow users to tweet in pictures of current tattoos, explore artists and tattoo shops in their area, and source inspiration for upcoming projects.

Bathroom Stall

Completed in 14 hours at a MakerSquare weekend Hackathon, Bathroom Stall allows Facebook authorization and integrates with the Facebook API to allow for reviews of people you love or people you love to hate. Take any Facebook URL to populate a friend, acquantaince, or stranger's public FB information.

Biblio

An exercise in creating a :has_many, :through ActiveRecord relationships. The home index shows all books added by all users (and populates information from the Goodreads API using the book's ISBN number). Each user page shows all books on that user's digital bookshelf. When a user destroys an entry from their own bookshelf, it still persists in the books database and on other users' bookshelves.

Gif-a-tweet

My first native Ruby application that I ported to Rails as my three-week checkpoint project at MakerSquare. Enter a twitter handle to generate a random tweet from the user's timeline. After eliminating stop words, two keywords are chosen to generate a random GIF using the Giphy API. Sometimes the results make no sense, but every once in a while the GIF/Tweet combo is pretty hilarious.
